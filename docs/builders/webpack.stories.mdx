## Использование с Webpack 5
Для корректной сборки кита при использование webpack 5 и выше необходимо воспользоваться расширением конфига сборки.

Конфигурация до расширения:
```js
// webpack.config.js
module.exports = {
    entry: {
        main: './src/index.js'
    },
    output: {
        path    : './dist',
        filename: '[name].bundle.js'
    },
    ...
};
```

Конфигурация после расширения:
```js
// webpack.config.js
const { extendWebpackConfig } = require('@kritua/ui-react/builders');

module.exports = extendWebpackConfig({
    entry: {
        main: './src/index.js'
    },
    output: {
        path    : './dist',
        filename: '[name].bundle.js'
    },
    ...
});
```
