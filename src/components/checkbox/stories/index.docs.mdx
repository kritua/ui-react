## Checkbox
Компонент `checkbox` соответствующий дизайну и его состояния.
Имеет максимально нативный API для беспроблемной работы c библиотеками управления формами такие как `react-hook-form`, `formik`, `final-form` и т.д.

## Использование
Импорт компонента в код.
```tsx
import { Checkbox } from '@kritua/ui-react';
```
